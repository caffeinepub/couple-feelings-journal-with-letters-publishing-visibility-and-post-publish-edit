# Specification

## Summary
**Goal:** Apply a romantic visual theme, allow fully open public letter access, and add inline letter expansion on the Public Letters page.

**Planned changes:**
- Apply a warm romantic color palette (rose, blush, cream, gold) with elegant serif/script headings and subtle heart/floral accents across all pages and components
- Remove all authentication guards from the Public Letters route so any visitor (logged in or not) can read, write, and submit public letters; update backend to allow anonymous submissions and reads
- Add inline "Read More" / "Read Less" toggle on each public letter card with smooth expand/collapse animation, no page navigation
- Add a romantic hero background image (rose petals, warm bokeh, blush/gold tones) to the Public Letters page header

**User-visible outcome:** Any visitor can browse, read, write, and submit public letters without logging in. The entire app has a romantic rose/gold aesthetic, and letter cards expand inline when "Read More" is clicked.
